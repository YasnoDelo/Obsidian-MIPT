Настройка ip-адресса:
`ip 192.168.13.11/24 192.168.13.1`

где:

| Часть команды   | Описание          |
| --------------- | ----------------- |
| `192.168.13.11` | Адресс устрйства  |
| `/24`           | Маска по CIDR     |
| `192.168.13.1`  | Дефолтный гейтвей |
|                 |                   |
